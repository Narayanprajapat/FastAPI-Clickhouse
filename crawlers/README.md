## Installation & Setup Instructions

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/Narayanprajapat/FastAPI-Clickhouse
    cd FastAPI-CRUD
    cd crawerls
    ```

2.  **Create .env for  crawerls/.env:**

    ```bash
    BEARER_TOKEN="
    ```

3.  **Run Command Compose:**

    ```bash
        poetry shell
        poetry install
        python3 sharpely_scrapper.py
    
    ```
